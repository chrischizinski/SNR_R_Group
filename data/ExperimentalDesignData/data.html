Data Sets

The data sets that are used for the main worked examples are available here, so you can work the the files and understand how your particular statistics package handles a particular analysis. In some cases, the raw data were included in the published paper, but, more commonly, the raw data were generously made available to us by the authors of the work. Please remember that in these cases, the data remain the intellectual property of those authors.

Each data set is listed by its subject, corresponding to the listings in the book index under "examples", and "examples, Worked". For example, Quinn's 1988 study on pulmonate limpets, used to illustate a two factor fixed effects ANOVA (Box 9-4) is listed as limpets, effects of season and adult density on fecundity. For each data set, we also describe its main use in the book, and include 3 files. A contents file lists the variables used in the study, their names in the data files, and the units of measurement. The data are contained in two files, suitable for importing into most statistical packages. There is a Microsoft Excel file, and a comma-delimited text (CSV) file. If your software does not read Excel files, use the CSV version. You should note that the files include the variable names as the first line. If your software does not accept this format, you will need to remove the first line before importing the file.
